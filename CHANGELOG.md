## [1.0.0] - 2026-03-29

### ✨ 新增

- **4 代理流水线**
  - 采集代理：热点抓取 + 选题生成（10 个）
  - 编辑代理：文章撰写 + 视觉 AI 配图 + SEO 优化
  - 审核代理：3 层审核机制（数据/来源/内容）
  - 发布代理：排版 + 推送草稿箱 + 数据归档

- **智能交互模式**
  - 全自动模式：自动选择最优选题和框架
  - 交互模式：关键节点用户确认
  - 分步执行：单独执行任一步骤

- **3 层审核机制**
  - 数据校验：A/B/C/D 分级（可查证率）
  - 来源核实：S/A/B/C/D 分级（权威性）
  - 内容审核：标题/客观性/逻辑/合规检查

- **视觉 AI 配图**
  - 封面自动生成（900x383 微信标准）
  - 内文配图 3-6 张（1280x720）
  - 信息图/数据图/漫画插图多种风格

- **学习机制**
  - 人工修改学习（diff 分析）
  - Playbook 自动更新（每 5 次修改）
  - 风格偏好固化

- **效果复盘**
  - 阅读量/分享率/完读率分析
  - 优化建议生成
  - 数据自动回填 history.yaml

- **客户配置系统**
  - style.yaml：风格配置
  - history.yaml：发布历史
  - corpus/：历史文章语料
  - playbook.md：风格指南
  - lessons/：人工修改记录

- **排版主题**
  - professional-clean：专业简洁
  - tech-modern：科技风
  - warm-editorial：暖色编辑
  - minimal：极简黑白

## 版本说明

### 版本号规则

遵循语义化版本规范（Semantic Versioning）：

- **主版本号（Major）**：不兼容的 API 修改
- **次版本号（Minor）**：向下兼容的功能性新增
- **修订号（Patch）**：向下兼容的问题修正

### 发布周期

- **主版本**：每季度发布
- **次版本**：每月发布
- **修订版**：按需发布

### 支持政策

- **当前版本**：完全支持
- **上一版本**：关键 bug 修复
- **更早版本**：不再支持
