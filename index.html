📊 审核标准

等级 数据可查证率 来源权威性 处理方式
S 级 100% 官方文件/权威媒体 直接发布
A 级 ≥80% 主流媒体/行业报告 直接发布
B 级 ≥60% 一般媒体/自媒体 修改后发布
C 级 <60% 不可靠来源 重写或弃用

🚀 快速开始

# 1. 克隆仓库 git clone https://github.com/your-org/Z5-WeChat-SOP.git cd Z5-WeChat-SOP # 2. 安装依赖 pip install -r requirements.txt npm install -g @wenyan-md/cli # 3. 配置环境变量 export WECHAT_APP_ID="your_app_id" export WECHAT_APP_SECRET="your_app_secret" export BAILIAN_API_KEY="your_api_key" # 4. 创建客户配置 cp clients/demo/style.yaml clients/your-client/style.yaml # 5. 运行全流程 python3 scripts/main.py --client your-client --mode auto
